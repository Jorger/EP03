Cifrado de Alberti
=========

El cifrado [alberti], se realiza haciendo coincidir una letra del disco exterior con la letra que se desee del disco interior. Esta pareja de caracteres son la clave que tanto emisor y receptor deber√°n conocer para llevar a cabo el proceso de cifrado/descifrado.

El texto en claro se cifra letra por letra haciendo coincidir las letras del mensaje en claro con el disco exterior y sustituyendolas por las letras correspondientes en el disco interior.
